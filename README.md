# 📓 痞老板日记本

> 简单好用的 Markdown 记事本，解决微软记事本存在的问题

## ✨ 功能特点

- 📝 **Markdown 支持** - 标题、列表、代码块、链接、图片等
- 📚 **多笔记本管理** - 创建多个笔记本，分类管理笔记
- 📑 **自动目录** - 根据标题自动生成目录导航
- 💾 **自动保存** - 数据自动保存到浏览器本地存储
- 🔄 **备份恢复** - 支持导出/导入 JSON 格式数据
- 🎨 **多主题切换** - 6 种精美主题任选
- 📁 **文件夹备份** - Chrome 浏览器支持备份到本地文件夹

## 🚀 使用方法

1. 下载 `index.html`
2. 用浏览器打开即可使用
3. 推荐使用 Chrome 浏览器

## ⌨️ 快捷操作

| 操作 | 说明 |
|------|------|
| `Ctrl + Enter` | 完成标题编辑并渲染 |
| 双击笔记本名称 | 重命名笔记本 |

## 🛠️ 技术特点

- 纯原生 HTML/CSS/JavaScript
- 零依赖，单文件运行
- localStorage 持久化存储
- IndexedDB 文件系统备份支持
